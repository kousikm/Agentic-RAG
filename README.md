Features
1. Configuration
Enter your OpenAI API Key in the sidebar. This key is used for both embedding documents and generating answers.
2. Data Ingestion
Local Files: Upload PDF or TXT files using the file uploader in the sidebar. Click "Process Uploaded Files" to ingest them.
Website: Enter a URL in the sidebar and click "Process Website" to ingest content from that page.
3. Chat Interface
Once data is ingested, use the chat interface to ask questions.
The agent will use the ingested documents/website content to answer your questions.
It uses Chroma DB for vectorization.
